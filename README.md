# 노무사 노동법 CBT V2.1 — 최근 5개년 업데이트

## 수록 범위
- 제30회(2021)~제34회(2025)
- 노동법Ⅰ·노동법Ⅱ 총 310문항
- 실제 기출 회차별·과목별·파트별 풀이
- 공식 확정답안 기준 채점
- 기출 선지 분해 복습문제 900문항 이상
- 오답노트·재풀이·파트별 통계

## 데이터 상태 표시
- `텍스트 원문 추출·공식답안 대조`: 검색 가능한 텍스트 문제
- `공식 스캔 원문 이미지 제공·텍스트 전환 검수 예정`: 원문 이미지를 보고 답하는 문제
- `공식 원문 이미지 보완`: 텍스트 추출이 불완전한 개별 문항
- `현행법 보정`: 개정 내용을 반영한 경우에만 별도 표시하도록 구조화

스캔 이미지 문항은 공식 문제를 빠짐없이 제공하기 위한 방식입니다. 해당 문항의 상세 선지별 문장 해설은 텍스트 전환 검수 후 단계적으로 보강합니다.

## 기존 GitHub Pages 업데이트
1. 기존 앱에서 학습기록을 백업합니다.
2. 이 ZIP의 압축을 풉니다.
3. 압축을 푼 폴더 **안의 파일과 폴더 전부**를 기존 저장소 최상단에 업로드합니다.
4. 같은 이름 파일은 교체하고 `assets/past-images`, `data/past` 등 새 파일도 함께 올립니다.
5. Commit changes 후 기존 GitHub Pages 주소를 새로고침합니다.

저장소 최상단에는 `index.html`, `assets`, `data`, `icons`, `sw.js`, `manifest.webmanifest`가 보여야 합니다.
