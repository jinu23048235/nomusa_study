# 노무사 노동법 1차 PWA — 실제 기출 기능 추가판

## 이번 업데이트
- `실제 기출` 메뉴 추가
- 회차별·파트별 실제 객관식 풀이
- 정답 선지뿐 아니라 모든 선지의 개별 해설
- 회차·연도·문항번호·공식 정답·근거 표시
- 법 개정 시 `현행법 보정` 배지를 표시할 수 있는 데이터 구조
- 1차 검증 수록: 2024년 제33회 제2~4문, 2025년 제34회 제39~40문

## 기존 GitHub Pages에 업데이트하는 방법
1. 이 ZIP을 압축 해제합니다.
2. 기존 GitHub 저장소의 루트에 있는 아래 파일을 새 파일로 교체합니다.
   - `index.html`
   - `sw.js`
   - `manifest.webmanifest`
   - `README.md`
   - `icons` 폴더
3. GitHub 저장소에서 **Add file → Upload files**를 누릅니다.
4. 압축을 푼 내부 파일과 `icons` 폴더를 드래그합니다.
5. 같은 이름의 파일을 덮어쓴 뒤 **Commit changes**를 누릅니다.
6. 1~3분 뒤 기존 GitHub Pages 주소를 다시 엽니다. 주소는 바뀌지 않습니다.

## 아이폰에서 구버전이 계속 보일 때
PWA 서비스워커 캐시 때문에 이전 화면이 남을 수 있습니다.
1. Safari에서 GitHub Pages 주소를 열고 새로고침합니다.
2. 그래도 같으면 홈 화면의 기존 앱 아이콘을 삭제합니다.
3. 설정 → Safari → 고급 → 웹사이트 데이터에서 `github.io` 또는 해당 사이트 데이터를 삭제합니다.
4. Safari로 주소를 다시 연 뒤 공유 → 홈 화면에 추가합니다.

학습기록은 브라우저 저장소에 보관됩니다. 사이트 데이터를 삭제하면 기존 기록도 지워질 수 있으므로, 필요하면 앱의 `학습기록 백업`을 먼저 사용하세요.
